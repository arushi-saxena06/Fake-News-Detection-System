📰 Fake News Detection using Machine Learning
📌 Problem Statement

Fake news spreads rapidly across digital platforms, making it difficult to distinguish between real and misleading information. This project aims to classify news articles as Fake or Real using Machine Learning techniques.

💡 Proposed Solution

The system uses:

Text Preprocessing

TF-IDF Feature Extraction

Logistic Regression Classifier

The trained model predicts whether a news article is Fake or Real.

⚙️ Technologies Used

Python

Scikit-learn

Pandas

NumPy

Streamlit

📊 Model Performance

Accuracy Achieved: 98.93%
